# LibrePay

A app to spread the \(financial\) freedom to small merchants. 

## Dependencies
This project uses [Xamarin][xamarin-docs] to create a app that runs natively on any
portable device.
So please refer to the [requirements wiki page][wiki-requirements]

## Building
